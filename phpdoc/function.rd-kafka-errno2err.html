rd_kafka_errno2err

(PECL rdkafka >= 0.9.1)

rd_kafka_errno2err — Convert the system errno to a kafka error code

Description

integer rd_kafka_errno2err ( integer $errnox )

Converts the system errno errnox to a kafka error code.

Parameters

errnox (integer)

A system errno

Return Values

Returns a kafka error code as an integer.
