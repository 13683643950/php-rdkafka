RdKafka::getOutQLen

(PECL rdkafka >= master)

RdKafka::getOutQLen — Get the out queue length

Description

public integer RdKafka::getOutQLen ( void )

Returns the current out queue length.

The out queue contains messages waiting to be sent to, or ackknownledged by, the broker.

Parameters

This function has no parameters.

Return Values

Returns the number of messages in the out qeueue.
