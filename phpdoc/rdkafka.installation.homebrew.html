Installing the extension on OSX with Homebrew

If you are using » Homebrew, the PHP tap includes formulae for installing the driver on various PHP versions.

  • php54-rdkafka
  • php55-rdkafka
  • php56-rdkafka
  • php70-rdkafka

For example, you might install the extension for PHP 7.0 using the following command: 

$ brew install php70-rdkafka
