(PECL rdkafka >= 1.0.0, librdkafka >= 0.9)

RdKafka\KafkaConsumer::getSubscription — Return the current subscription

Description

public array RdKafka\KafkaConsumer::getSubscription ( void )

Return the current subscription as set by RdKafka\KafkaConsumer::subscribe()

Parameters

This function has no parameters.

Return Values

Returns an array of topic names
